# McFlourish

**McFlourish** is a command-line tool (CLI) that detects **smoke and fire in real-time** using AI.  

We are building an open-source project and need help from the community: **contribute indoor photos** of fire, smoke, and normal environments to improve the model.  

Your contributions will help make McFlourish more accurate and reliable for real-time detection.  

**Get started:**  
- Train the model: `python train.py`  
- Run real-time detection: `python main.py` 

**Important:**
- Code usage is restricted. You may only replicate functionality independently without viewing the source.
- Dataset usage is restricted. You must have permission from the contributors before using or sharing images.
